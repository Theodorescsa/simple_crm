Các api CRUD đều cần token để xác thực

Các API hỗ trợ

  • API Đăng ký người dùng:

    Đăng ký tài khoản mới trên hệ thống.

    POST /authentication/signup/
  • API Xác thực:

    Đăng nhập và nhận token xác thực.

    POST /authentication/signin/
  • API Quản lý khách hàng:

    Tạo, chỉnh sửa, và xóa khách hàng.

    GET/POST/PUT/DELETE /crm/customers/
  • API Quản lý sản phẩm:

    Tạo, chỉnh sửa, và xóa sản phẩm.

    GET/POST/PUT/DELETE /crm/products/
  • API Quản lý nhân viên:

    Tạo, chỉnh sửa, và xóa nhân viên.

    GET/POST/PUT/DELETE /crm/employees/
  • API Quản lý bảng công việc, nhiệm vụ:

    Tạo, chỉnh sửa, và xóa bảng công việc, nhiệm vụ.

    GET/POST/PUT/DELETE /crm/task-board/
